What is it?

TipiCalc Calculator is a lightweight, fast, keyboard‑first calculator. Open it from the extensions toolbar or with a hotkey (default: Alt+T), type your expression, and press Enter — no mouse needed.

Get it from Chrome Web Store: TipiCalc Calculator

Features

  • Stores history of expressions
  • Saves the entered expression when the window is closed
  • Supports multiple functions and operators
  • Time arithmetic mode: prefixes !time/!t/time/t, optional colon, minutes‑only terms
  • Mode‑aware insertion from history: clicking a time result inserts time <value> into empty input
  • Last result placeholder @
  • Auto‑prepend of the last result when you start with an operator
  • Open it from the toolbar or by shortcut (Alt+T)
  • Friendly to both dot and comma decimals (outside function calls)
  • Works offline

Operators: +, −, *, /, integer division (// or \), remainder (%), exponent (** or ^)

Constants: pi, e

Functions: sqrt, root, ln/log, lg/log10, log(x, base), sin/cos/tan, asin/acos/atan, rad2deg, deg2rad, max, min, sum, mul, floor, ceil, round, trunc/int, frac, abs, random/rand/rnd

Conveniences:

  • Insert the last successful result with @: @ * 2, max(@, 10)
  • Start with a binary operator and TipiCalc will auto‑use the last result: typing + 5 becomes <last> + 5
  • Outside functions you can use either dot or comma: 3.14 or 1,5 (inside functions — use dots)

Examples

  • sqrt(9) → 3
  • root(3, 8) → 2
  • ln(e) → 1
  • lg(1000) → 3
  • log(8, 2) → 3
  • sin(pi/2) → 1
  • 7 // 3 → 2
  • frac(3.14) → 0.14
  • @ * 2 — previous result * 2

Time arithmetic mode

  • time 10:50+18 → 11:08
  • t -1:00 + 0:30 → -0:30
  • !t 2:00 - 0:30 → 1:30
  • time 18 + 1:00 → 1:18
  • Invalid: time 10:5 — minutes must be two digits

Version history

v1.3.0 — 2025-11-12

  • Time arithmetic improvements: prefixes !time/!t/time/t; colon after the prefix is optional
  • Minutes‑only terms allowed in any position (including the first); result formatted as [-]H:MM
  • Strict H:MM minutes: exactly two digits 00..59 (e.g., 10:5 is invalid)
  • Mode‑aware history insertion: time results insert with time prefix when the input is empty

v1.2.0 — 2025-11-01

  • Persisted the last typed expression between popup sessions

v1.1.1 — 2025-10-28

  • Adjusted internal logging so invalid expressions don’t count as errors

v1.1.0 — 2025-10-27

  • Fixed precedence of exponent (^) and unary minus
  • Corrected calculations with @ when its value is positive
  • New functions: gcd, lcm, avg, is_prime
  • Extended: random
  • Operator additions: synonym : for division, support for unary plus
  • Improved history selection when it’s full
